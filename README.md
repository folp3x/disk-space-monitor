# Disk Space Monitor

Консольное клиент-серверное приложение для удаленного мониторинга дискового пространства.

Клиент отправляет путь к файлу или директории на сервер, а сервер возвращает информацию о свободном и занятом месте на соответствующем диске.

---

# Возможности

## Клиентская часть

- Ввод пути к файлу или директории на удаленном сервере
- Отправка UDP-запроса
- Прием бинарного ответа (2 числа uint64_t в Big-Endian)

## Серверная часть

- Многопоточная обработка запросов
- Использование `statvfs` для анализа файловой системы
- Расчет:
  - свободного места: `f_bavail * f_frsize`
  - занятого места: `(f_blocks - f_bfree) * f_frsize`
- Отправка результата в виде 16-байтного бинарного ответа
- Опциональная диагностика `/proc/mounts` (DEBUG)

---

# Протокол взаимодействия

## Запрос (Client => Server)

- Тип: строка
- Содержимое: путь к файлу или директории
- Ограничение: <= 4095 символов

## Ответ (Server => Client)

- Тип: бинарные данные (16 байт)
- Структура:
  - uint64_t free_space
  - uint64_t used_space
- Порядок байт: Big-Endian

# Сборка и запуск

## Сервер

```bash
g++ -std=c++11 server.cpp -o server -lpthread

./server 8080
```

### Debug-режим

```bash
g++ -std=c++11 -DDEBUG server.cpp -o server -lpthread
```

---

## Клиент (MinGW)

```bash
g++ -O2 -std=c++11 client.cpp -o client.exe -lws2_32

client.exe 127.0.0.1 8080
```
